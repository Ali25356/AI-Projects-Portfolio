# Arabic NLP Sentiment & Slang Analysis

## Description
An Arabic NLP project for classifying Egyptian/Arabic text into sentiment categories.

## Features
- Arabic text cleaning
- TF-IDF feature extraction
- Machine learning classifier
- Evaluation report
- Prediction function for user input

## Run

```bash
pip install -r requirements.txt
python arabic_nlp_classifier.py
```

## CV Description
Developed an Arabic NLP system for sentiment analysis and informal slang interpretation using text preprocessing and machine learning classification.
