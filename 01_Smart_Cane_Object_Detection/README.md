# Smart Cane Object Detection Demo

## Description
A computer vision project for assistive navigation.  
The system detects objects in real time using YOLO and gives audio feedback.

## Features
- Real-time object detection
- Webcam/video support
- Audio feedback
- Clean code for future Raspberry Pi integration
- Suitable for blind-assistance / smart cane project

## Run

```bash
pip install -r requirements.txt
python smart_cane_demo.py
```

## CV Description
Developed a smart assistive navigation system using YOLO object detection and audio feedback for real-time obstacle awareness.
