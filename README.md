# AI Projects Portfolio for Job Fair

This repository contains 3 AI projects suitable for an Artificial Intelligence student portfolio:

1. **Smart Cane Object Detection Demo**  
   Real-time obstacle/object detection using YOLO/OpenCV with optional audio feedback.

2. **Image Classification using Transfer Learning**  
   CNN image classification project using MobileNetV2.

3. **Arabic NLP Sentiment & Slang Analysis**  
   Arabic/Egyptian text classification pipeline for sentiment or slang understanding.

Each project includes:
- Clean Python code
- Jupyter Notebook
- README file
- Requirements file
- GitHub-ready structure
